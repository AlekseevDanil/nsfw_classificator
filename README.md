# nsfw_classificator

* _inspector_model.h5_ - обученная модель
* _start_classification()_ - класс который производит классификацию видео

***Как все работает?***

main.py - скрипт в котором находиться класс. В этот класс мы передаем 1 обязательный и 1 необязательный параметры. Первый параметр путь к нашему видео файлу, который нужно проклассифицировать. Второй параметр означает поть до файла модели, по умолчанию наша основная модель (вдруг мы хотим протестировать на своей или переобучили модель).

После того, как мы вызвали класс и правильно передали все параметры, функция загружает модель обозначает 3 параметра (Нейтрально, Порно и Сексуально) нулями, после чего загружает наше видео. После этого, функция делает захват каждого кадра (уменьшает его, переводит в числа), и передает в модель, для ответа от 0 до 1. Таким образом для каждого параметра мы собираем все значения и делим на колличество (находим среднее значение) и функция возвращает 3 параметра (среднее значения каждого в видео, 2 знака после запятой) - ответ на наш вопрос (Neutral, Porn, Sexy), чье значение выше, то и преобладает в видео
